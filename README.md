## ✨ Funkcjonalności
- **Tworzenie Notatek:** Możliwość szybkiego zapisywania tekstowych notatek.
- **Interfejs Użytkownika:** Wykorzystanie standardowych komponentów Android XML do budowy intuicyjnego UI.
- **Zarządzanie Danymi:** Obsługa stanów aplikacji i przechowywanie informacji o notatkach.
- **Natywne działanie:** Aplikacja zoptymalizowana pod kątem wydajności na urządzeniach z systemem Android.
